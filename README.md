# `dashboard_data`

Data voor het AMRZNN-dashboard. Deze data zijn anoniem en publiek toegankelijk, zolang voldaan wordt aan de juridische voorwaarden.

Deze gegevensrepository is gelicentieerd onder de [Creative Commons Naamsvermelding-NietCommercieel-GelijkDelen 4.0 Internationaal (CC BY-NC-SA 4.0)](https://creativecommons.org/licenses/by-nc-sa/4.0/deed.nl) licentie. Dit betekent dat elke gebruiker of ontwikkelaar, in elke vorm dan ook:

- **Naamsvermelding** moet geven, op een passende manier, een link naar de licentie moet plaatsen en moet aangeven of het werk is gewijzigd sinds het van deze repository overgenomen is
- De data **niet commercieel** mag gebruiken
- Bewerkingen of afgeleide werken alleen mag verspreiden onder **dezelfde licentie** (CC BY-NC-SA 4.0)
- **Geen aanvullende beperkingen** mag opleggen die anderen juridisch beperken in het gebruik dat de licentie toestaat

Gebruik van de data in deze repository is uitsluitend toegestaan onder de voorwaarden van deze licentie **én de aanvullende gebruiksvoorwaarden** zoals uiteengezet in [Gebruiksvoorwaarden](TERMS_OF_USE.md). Deze voorwaarden stellen dat de data beschikbaar gesteld worden **zonder enige uitdrukkelijke of stilzwijgende garantie** met betrekking tot juistheid, volledigheid, actualiteit of geschiktheid voor een bepaald doel (Artikel 7).
